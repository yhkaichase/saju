# 사주(四柱) 도메인 정확성 규칙

> 이 도메인은 **정확성이 최우선**입니다. 아래 규칙을 추정으로 구현하지 말고,
> 항상 근거/출처를 코드 주석과 PR 본문에 남기세요. 모호하면 구현 전에 질문하세요.

## 기본 개념

- 사주(四柱)는 연주·월주·일주·시주 4개의 기둥이며, 각 기둥은
  **천간(天干, heavenlyStem) + 지지(地支, earthlyBranch)**로 이루어집니다.
- 천간 10개: 甲乙丙丁戊己庚辛壬癸 (gabeul…)
- 지지 12개: 子丑寅卯辰巳午未申酉戌亥
- 60갑자(六十甲子)는 천간·지지의 최소공배수 조합.

## 반드시 근거가 필요한 계산

1. **음양력 변환 (양력 ↔ 음력)**
   - 검증된 만세력/천문 데이터 또는 신뢰할 수 있는 라이브러리를 사용합니다.
   - 라이브러리 사용 시 **출처·버전·정밀도(분 단위/시간대)**를 기록합니다.
   - 한국 표준시(KST, UTC+9) 기준인지, 진태양시 보정 여부를 명시합니다.

2. **연주(年柱) 결정**
   - 새해 기준은 양력 1/1도 음력 설날도 아닌 **입춘(立春)**입니다.
   - 입춘 시각(분 단위) 경계 처리를 정확히 합니다.

3. **월주(月柱) 결정 — 절기 기준**
   - 월의 경계는 음력 초하루가 아니라 **12절기(節氣)**입니다.
   - 예: 인월(寅月)은 입춘부터, 묘월(卯月)은 경칩부터.
   - 월간(月干)은 연간(年干)에 따라 결정됩니다(오호둔/五虎遁).

4. **일주(日柱) 결정**
   - 60갑자 연속 순환. 기준일(epoch)을 명시하고 검증합니다.

5. **시주(時柱) 결정 — 야자시/조자시**
   - 하루의 경계와 자시(子時, 23:00~01:00) 처리 방식을 **명시적으로** 정합니다.
     - 야자시(夜子時, 23:00~24:00) / 조자시(朝子時, 00:00~01:00) 처리 정책을
       코드와 문서에 분명히 기록합니다.
   - 시간(時干)은 일간(日干)에 따라 결정됩니다(오서둔/五鼠遁).

## 파생 계산

- **오행(五行)**: 천간·지지의 목화토금수 분류와 생극(生剋) 관계.
- **십신(十神)**: 일간 기준 다른 간지와의 관계.
- **대운(大運)**: 성별·연간 음양에 따른 순행/역행, 시작 나이(대운수) 계산.

## 테스트 원칙

- **알려진 명식(命式) 사례를 골든 테스트로 고정**합니다.
  - 절기 경계일(입춘 당일 등), 자시 경계(23:30, 00:30), 윤달 케이스를 반드시 포함.
- 외부 만세력 서비스/책과 교차 검증한 결과를 테스트 데이터 출처로 남깁니다.
- 계산 로직 변경 시 골든 테스트가 깨지면 **원인을 규명하기 전까지 값을 갱신하지 않습니다**.
